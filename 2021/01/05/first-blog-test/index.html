基于hexo和github搭建个人网站

Hexo: 快速简洁且高效的博客框架，可以生成静态网页。

Git-hub:提供静态网页托管服务 。

参考博文：

https://www.cnblogs.com/shwee/p/11421156.html

https://hexo.io/zh-cn/docs/setup

https://segmentfault.com/a/1190000017986794

常用命令

hexo new “标题” // 在/source/_posts 下新建标题.md的文件

hexo g // 生成静态网页

hexo s // 本地预览静态网页

hexo d // 部署到git hub 上
