立学6-百万up主进阶之路

百万up主第一步–IMovie

百万Up主第二步–PreMiere

Pr教程0基础急速上手视频制作!UP主必备

新建项目-名称-保存项目的地方【放在一个大的文件夹里】

1.主界面–效果控件–左上角、导入素材–左下角、预览素材–右上角、深度加工–右下角

2.导入素材–右键-新建素材箱、导入单个文件

->简单编辑–要编辑素材的时候，拖到右下角的深度加工框中，右上角的浏览框可以查看效果

->截取视频长短：1.剃刀工具 (在素材条上点击) 2. 直接拉编辑素材的边缘长度【和Imovie差不多】

-> 删除视频片段： 1.直接选中状态回车 2.右键删除

->素材条尺度缩放：深度加工窗口有一个放大镜， 可以用其放大素材条的跨度，使该条便于拖动截取。缩放回来alt 就会变成减号，就可以缩小回去

-> 一个界面两个视频窗口：视频条直接叠加，在窗口中调节视频窗口的大小。

-> 倍数播放：视频条右键-速度/持续时间-调整速度百分比-注意点：变快视频条缩短无影响，变慢视频条变长，需要后来下一个视屏，在编辑条上预留一定的空间，避免丢失信息。

->转场效果：左下角-效果-视频过渡-拖动到需要添加专场效果的地方

->视频段的复制control + c -> control + v

->左上角-播放效果-大括号-可以标记入点和出点-然后拖动视频到右下角，可以实现片段截取加入

3.背景音乐–导入音频素材，【和Imovie差不多，拖到相应的视频片段下面就可以了】

左边的工具栏框里头 有A1，A2， A3 三种不同的通道

S-独奏-会加入背景音，M-静音-背景音里不会有。【可以利用这个来作背景音乐的最优选择】

音乐编辑条左侧锁住，就不会被其他片段音乐覆盖

4.字幕-工具栏-

静态字幕-可以随意添加在屏幕的任何位置（很方便呀）右边可以进行字幕的设计， 设计完了就会成为素材库里的一个素材，只要拖入到对应的位置就可以了。

滚动字幕–字幕编辑页面-滚动/游动选项

5.利用关键帧做一个高仿弹幕

字幕-静态字幕-编辑文件-关闭保存字幕素材-添加到视频编辑条的指定位置-点击出现左上角的深度加工窗口 –关键帧-fx 调整坐右边的位置，可以实现字幕的移动

6.鬼畜教学–先导入音乐-卡音乐节奏放入视频片段

7.片尾字幕-由大变小 右边出现滚屏

8.视频导出-文件导出媒体 格式： H.264 - mp4的文件 预设：匹配源-高比特率-导出视频-导出音频
